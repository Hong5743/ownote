# Ownote
>Ownote 사원들만을 위한 사내 그룹웨어, 출 퇴근 관리 및 주소록을 통한 타 부서와의 협업 능률 향상, 사내 동아리 설립을 통한 유대감 형성, 매달 초 게시판 활동을 통한 회식비 지원 이벤트를 통한 사원들의 애정도 상승 효과를 기대하는 이 모든 것을 포함한 사내 그룹웨어 입니다.
